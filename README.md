Projeto feito com nextJs <3

Para iniciar o projeto: yarn dev

Observações:

- Alguns personagens não possuem thumbs na api. Fiquei em dúvida se eu limitava a somente personagens mais famosos, porém resolvi deixar o site mostrando todos.

Utilização:
Basta clicar em um personagem para saber mais informações sobre ele. Os quadrinhos no qual ele apareceu são mostrados embaixo e também são clicáveis, caso se queira saber mais informações.

A busca retorna personagens com nomes parecidos. (A busca deve ser em inglês)
